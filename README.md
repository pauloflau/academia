# Projeto Academia
Projeto Java com **Spring Boot** para desenvolvimento de um Sistema de gerenciamento de academia com separação entre **backend** e **frontend**.

---

## Tecnologias Utilizadas

**Backend**
- Java 17
- Spring Boot

**Frontend**
- Em definição

Possíveis tecnologias:
- HTML, CSS e JavaScript puro
- React
- Angular

**Banco de Dados**
- H2
- PostgreSQL(Futuramente)

---

## Como rodar o projeto
**Backend (Spring Boot)**

1. Clone o repositorio:
```json
git clone https://github.com/seu-usuario/academia.git
cd academia
```
2. Configure o banco de dados no application.properties ou application.yml.
3. Rode a aplicação com:
```json
./mvnw spring-boot:run
```
4. API estará disponível em:
```json
http://localhost:8080
```

Ainda não implementado.
